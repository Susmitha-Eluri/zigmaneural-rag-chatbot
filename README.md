## ğŸ› ï¸ Project Installation and Setup

To run the ZigmaNeural Assistant as a Chrome Extension:

1.  **Download project**: Clone or download this repository.
2.  **Open Extensions**: In Google Chrome, navigate to `chrome://extensions`.
3.  **Enable Developer Mode**: Toggle the switch in the top-right corner.
4.  **Load Extension**:
    - Click the **"Load unpacked"** button.
    - Select the **`rar_frontend/dist`** folder from the downloaded files.
5.  **Start Chatting**: Click the ZigmaNeural icon in your extension bar to open the sidebar on any webpage.
